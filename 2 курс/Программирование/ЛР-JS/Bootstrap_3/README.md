# Постановка задачи
Создайте новый проект (новую папку), используя переводчик или без него выполните [шаги по сборке Bootstrap-приложения с использованием сборщика Vite](https://getbootstrap.com/docs/5.3/getting-started/vite/). 

Интегрируйте в эту страницу скрипт, обновляющий время с помощью luxon, реализованный на [Этапе 1](https://moodle.herzen.spb.ru/mod/url/view.php?id=1301429 "https://moodle.herzen.spb.ru/mod/url/view.php?id=1301429"). Не забудьте установить необходимые зависимости.

Реализуйте аналогичный "дизайн", созданный на этапе 2 (колонки, кнопка, модальное окно). Постарайтесь минимизировать использование компонент из бутстрапа, чтобы бандл получился как можно меньше.
